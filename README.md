# Product Management System (MERN CRUD)

using:

* MongoDB  
* Express.js  
* React (Vite)  
* Node.js  



---

## Features

* Add new product  
* View all products  
* Update product details  
* Delete product  
* Mobile friendly UI  
* Backend API with MongoDB  

---

## Backend Run:

* npm run dev

## frontEnd Run:

* npm run dev
